🔐 Password Generator

A simple and interactive Password Generator Web App built using HTML, CSS, and JavaScript. This tool allows users to generate strong and secure passwords based on customizable options like length and character types.

🚀 Features
Generate secure random passwords instantly
Adjustable password length using a slider
Option to include:
* ✅ Lowercase letters (a-z)
* ✅ Uppercase letters (A-Z)
* ✅ Numbers (0-9)
* ✅ Symbols (@, -, *, etc.)
Copy password to clipboard with one click
Clean and user-friendly UI
🛠️ Tech Stack
* HTML5 – Structure of the application
* CSS3 – Styling and layout
* JavaScript (Vanilla JS) – Functionality and logic

⚙️ How It Works
1. Select the desired password length using the slider
2. Choose the character types you want to include
3. Click the "Generate Password" button
4. Copy the generated password using the copy icon
